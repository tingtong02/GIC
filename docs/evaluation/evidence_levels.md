# Evidence Levels

## Level 1

Direct device-level GIC measurements. This level supports numeric error statements such as MAE or RMSE against measured targets.

## Level 2

A small set of local devices or regional measurements with meaningful partial alignment. This level supports local numeric or peak-based validation, but not whole-grid accuracy claims.

## Level 3

Event-level trend, peak timing, or ranking references. This level supports statements about temporal alignment, peak coincidence, or relative risk ordering.

## Level 4

Physical plausibility or event-consistency only. This level supports qualitative consistency statements and cautionary discussion, not accuracy claims.

## Reporting Rules

- Never mix Level 1-4 into a single accuracy score.
- Never describe Level 3 or Level 4 agreement as exact prediction accuracy.
- Always state limitations and mapping assumptions per event.
